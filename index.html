IGDB credentials are set on the server, not here. They live as environment variables in your Netlify dashboard so they never reach the browser.
  1. Sign in at dev.twitch.tv/console and register an application (OAuth Redirect: http://localhost, category: Application Integration)
  2. Copy the Client ID; click "New Secret" to generate a Client Secret
  3. In Netlify: Site settings → Environment variables → add TWITCH_CLIENT_ID and TWITCH_CLIENT_SECRET
  4. Trigger a redeploy in Netlify (Deploys → Trigger deploy)
