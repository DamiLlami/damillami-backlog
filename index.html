IGDB setup — credentials live as Netlify environment variables.
  1. Sign in at dev.twitch.tv/console, register an application (OAuth Redirect: http://localhost, category: Application Integration)
  2. Copy the Client ID; click "New Secret" for a Client Secret
  3. In Netlify: Site configuration → Environment variables → add TWITCH_CLIENT_ID and TWITCH_CLIENT_SECRET
  4. Trigger a redeploy
Free tier: 60,000 requests/month.
RAWG setup
  1. Get a free key at rawg.io/apidocs
  2. In Netlify: add env var RAWG_API_KEY
  3. Trigger a redeploy
Free tier: 20,000 requests/month.
OMDb setup — gives Rotten Tomatoes scores for movies, IMDb ratings for both.
  1. Get a free key at omdbapi.com/apikey.aspx (1,000 requests/day on free tier; $1/mo for unlimited)
  2. In Netlify: add env var OMDB_API_KEY
  3. Trigger a redeploy
Note: OMDb only returns Rotten Tomatoes scores for movies. TV shows get IMDb rating instead.
